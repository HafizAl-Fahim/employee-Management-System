Project Details

This web application has been prepared in order to attain the requirement of the project for the Object-Oriented Programming II Lab course. As part of the course curriculum of OOP Lab of CSE, Daffodil International University, this project has been erected and submitted. The main reason of this project set by OOP II Lab is to attain overall knowledge that participants understand from this course which is utilized to solve some real-life problems.

So, I Hafiz Al Fahim tried to exhibit the real situation of this web application. Which is an innovative idea. The web application is developed with Django Framework. The web application is called ‘Employee Management System’. It is a very demandable web application nowadays for collecting data.
